Sobre mi

Dejame presentarme.

Diseñador Gráfico y desarrollador Fullstack con amplios conocimientos tanto de Backend como Frontend, mis herramientas de trabajo como diseñador son PHOTOSHOP, ILLUSTRATOR, CINEMA 4D y AFTER EFFECTS, entre otros. Como desarrollador web mis conocimientos abarcan el uso de lenguajes y frameworks tales como RUBY ON RAILS, POSTGRESQL, GIT, HTML5, CSS3, JAVASCRIPT y JQUERY. Estas herramientas y conocimientos me permiten entregar un servicio de alta calidad en el desarrollo de paginas web estaticas o dinamicas.
